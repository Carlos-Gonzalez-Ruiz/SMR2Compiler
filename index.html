Online SMR2 Compiler

Online compiler and emulator to generate and run binaries for ProcSMR2

Write the low level SMR2 code that you wish to compile to machine code for the microcontroller ProcSMR2. Once you think your code is ready to be executed, click on the compile button to get the resulting binary.

After the machine code has been generated, click on the Execute button to run it.
(Note that you can also write in the binary box. This is not highly recommended due to possible mistakes which may result in unexpected output)

See readme.txt file for further information about SMR2 language.
